Documentation for Button Project

Adding the App

  1. Navigate to the Button Project’s authorized download page at: https://0u9lz2ua.ngrok.app/api/zoomapp/install
  2. Log into Zoom
  3. Check off “Allow this app to use my shared access permissions.”
  4. Click the “Allow” button

Usage

  1. Press “Authorize” if the “Log in” button is greyed out
  2. Press the “Log in” button

Students

Joining a lecture

  1. Joining is done automatically when a recording is started

Pressing the button

  1. Press the button

Leaving a lecture

  1. Press the “Leave lecture” button

Professors

Starting a lecture

  1. Lecture sessions are started automatically on cloud recording

Ending a lecture

  1. Lecture sessions are ended automatically on end of cloud lecture

Removing the App

  1. On the Zoom client, navigate to “Apps” on the header
  2. On the Apps screen, press the “gear” icon on the top right
  3. Click the “Manage” link, which will redirect you to https://marketplace.zoom.us/user/installed
  4. Click the “Remove” button next to “Button Project”
